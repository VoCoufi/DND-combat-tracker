## Combat Mechanics Enhancements

### 1. Legendary Actions Tracking
**Priority**: ðŸ”´ High
**Difficulty**: Medium
**Estimated Effort**: 2-3 days

**Description:**
Track legendary action points for boss monsters, including remaining points, available actions, and automatic reset on turn start.

**Use Case:**
Boss fights are central to D&D. DMs need to track 3 legendary action points that reset at the start of the creature's turn. Without tracking, it's easy to forget how many remain or use too many.

**Technical Approach:**
- Add `legendary_actions` field to Combatant (optional)
- Store max points, current points, and action descriptions
- Auto-reset on turn start
- UI indicator showing "LA: 2/3" next to boss HP
- Menu to spend legendary actions (deduct points)

**Implementation Notes:**
- Add to combatant creation flow (optional field)
- Display alongside reactions if creature has them
- Could share UI with reaction tracking
